# The arrival road

The road into a server: loaded cold, before anything else. It teaches the *order* — arrive → reconcile → act → land — and why the order is load-bearing (a correct entry authored from an unreconciled standpoint is a permanent, attributed, wrong trace). It carries the always-on disposition list, the dock index, and the verb vocabulary; the literal calls live in each dock.
