# The reconcile dock

The hinge before every act. Two moves, both required: pull what changed in canon and actually take it up; then self-report — honestly, attributed — what updated in *you*. An empty diff still reconciles; a reconcile is a cursor-sync, never a clock-tick. The dock carries the self-report quality discipline (the test: could this sentence have been written by anyone who read the same diff?).
